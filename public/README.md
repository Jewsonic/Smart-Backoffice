# 📁 Pasta Public - Arquivos Estáticos

Esta pasta contém arquivos estáticos que serão servidos diretamente pelo Vite.

## 🖼️ Favicon

**Para adicionar seu favicon personalizado:**

1. Coloque seu arquivo de imagem nesta pasta com o nome `favicon.jpg` (ou `favicon.png`)
2. O arquivo já está configurado no `index.html` para ser carregado automaticamente
3. Formatos suportados: `.ico`, `.png`, `.jpg`, `.svg`

**Recomendação:** Para melhor compatibilidade, use PNG ou ICO com 32x32 ou 64x64 pixels.

---

## ✅ Arquivos que podem ir aqui:
- Favicon (favicon.ico, favicon.png, favicon.jpg)
- Imagens estáticas
- Arquivos de fontes
- Robots.txt
- Sitemap.xml
